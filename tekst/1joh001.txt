===Første Johannesbrev, kapitel 1===

==Johannes skriver som øjenvidne om Jesus som livets ord==

v1 Det som var fra begyndelsen, det vi har hørt, det vi har set med vores egne øjne, det vi betragtede, og vores hænder følte på, nemlig om livets ord
v2 – og livet åbenbarede sig, og vi har set og vidnet og forkyndt jer det evige liv, som jo var hos Faderen og blev åbenbaret for os –
v3 hvad vi har set og hørt, forkynder vi også jer for at også I må have fællesskab med os; og vores fællesskab er også med Faderen og med hans søn Jesus Kristus.
